# SnowmanBlog 开发路线图

## 版本历史

### v1.7.0 (2026-04-30)
- Docker 支持：PHP 8.2-FPM + Nginx + MySQL + Redis + Supervisor（含定时任务队列）
- 自动化测试：Feature 测试覆盖前端、后台、认证等核心流程
- GitHub Actions CI：自动运行 PHPUnit + Laravel Pint 代码风格检查
- 贡献者文档：CONTRIBUTING.md、CHANGELOG.md、Issue/PR 模板
- 数据库自动备份：每日凌晨自动 mysqldump，保留 30 天自动清理
- 后台备份管理：Filament 页面支持手动备份、列表查看、下载、删除

### v1.6.12 (2026-04-29)
- 首页新增开源项目入口（Gitee、GitHub），带品牌 Logo 与跳转链接

### v1.6.11 (2026-04-29)
- 新增后台文章草稿自动保存（基于 localStorage，每 5 秒自动保存表单数据）
- 新建/编辑文章时检测到草稿可一键恢复或清除
- 兼容富文本编辑器、SEO 字段、发布时间等各类表单数据

### v1.6.10 (2026-04-29)
- 新增文章阅读进度条（文章详情页顶部固定渐变进度条，滚动实时更新）
- 进度条基于文章主体滚动位置计算，平滑过渡动画

### v1.6.9 (2026-04-29)
- 新增文章分享按钮（微信二维码、微博、Twitter、复制链接）
- 微信分享支持弹窗展示当前页面二维码
- 复制链接兼容 HTTP/HTTPS 环境

### v1.6.8 (2026-04-29)
- 新增文章点赞功能（基于 IP 防刷，同一 IP 只能点赞一次）
- 文章详情页增加粉色点赞按钮，显示当前点赞数
- 首页文章列表显示点赞数
- 后台文章管理列表新增「点赞数」列，支持排序

### v1.6.7 (2026-04-28)
- 评论/留言 XSS 安全防护：引入 HTMLPurifier 过滤用户提交的 content，自动清除恶意脚本标签
- 评论/留言支持 Markdown 语法渲染（粗体、斜体、代码块、列表、引用等）
- 评论/留言表单新增「支持 Markdown 语法和 Emoji」提示
- 为评论/留言 Markdown 内容增加独立的代码高亮与排版样式

### v1.6.6 (2026-04-27)
- 新增文章置顶功能（后台 Filament 文章管理可开关置顶）
- 首页/分类/标签/搜索结果列表优先展示置顶文章
- 置顶文章在列表页显示红色「置顶」标识

### v1.6.5 (2026-04-27)
- 新增评论回复邮件通知（评论收到回复时自动邮件通知原评论者）
- 新增热门文章排行榜侧边栏（文章详情页右侧展示浏览量 Top 5，Redis 缓存 10 分钟）
- 新增文章图片灯箱放大（medium-zoom 点击放大，支持深色背景）

### v1.6.4 (2026-04-27)
- 缓存策略优化：切换 CACHE_STORE 为 Redis，启用 Cache::tags 分组管理
- 首页/分类/标签文章列表增加分页缓存（5 分钟）
- 文章详情页相关推荐、上一篇/下一篇增加缓存（10 分钟）
- ViewComposer 全局设置与友链增加缓存（settings 24h / links 1h）
- 文章/设置/友链变动时自动清除对应 tag 缓存
- 图片懒加载：首页、文章页、相关文章封面图及文章内容图均启用 native lazy loading

### v1.6.3 (2026-04-27)
- 新增评论/留言防刷保护（同一 IP 每分钟最多 3 次提交）
- 新增重复内容检测（同一 IP 60 秒内不能提交相同内容）
- 前端表单按钮提交后自动禁用，防止重复点击

### v1.6.2 (2026-04-27)
- 搜索优化：MySQL FULLTEXT 全文索引（ngram 中文分词）
- 搜索兜底策略：FULLTEXT 无结果时自动降级到 LIKE
- 新增 `ft_posts_search` 全文索引覆盖 title、content、excerpt

### v1.6.1 (2026-04-27)
- 新增后台个人资料页（Filament Profile）
- 支持修改登录账号的姓名、邮箱、密码、头像

### v1.6.0 (2026-04-27)
- 新增多语言支持（zh_CN / en），基于 Laravel i18n JSON 语言文件
- 新增语言切换器（桌面端 / 移动端导航栏）
- 全站前端文案国际化（导航、搜索、主题切换、文章页、评论区等）
- 新增 SetLocale 中间件，支持 URL 参数与 Session 持久化切换语言

### v1.5.0 (2026-04-27)
- 新增友情链接管理（Filament 后台 CRUD + 首页展示）
- 新增暗色模式切换（localStorage 持久化 + Tailwind darkMode: class）
- 新增 SEO 结构化数据（首页 WebSite + 文章页 BlogPosting / BreadcrumbList）
- 新增数据库浏览量统计（IP + 日期去重防刷）
- 完善 README 安装教程

### v1.4.1 (2026-04-25)
- 修复评论/留言邮箱非必填数据库兼容性问题
- 修复后台中文化字段标签显示问题
- 修复博主自己回复仍需审核的 bug
- 修复留言板后台页面 500 错误
- 评论邮件通知使用正确的管理员邮箱

### v1.4.0 (2026-04-25)
- 新增 Sitemap 功能 (/sitemap.xml)
- 新增 RSS 订阅功能 (/rss.xml)
- 新增文章 SEO 自定义（Meta 标题、描述、关键词）
- 新增文章上一篇 / 下一篇导航
- 新增文章相关推荐
- 新增文章定时发布（Laravel Schedule + Cron）
- 新增评论邮件通知（QQ 邮箱 SMTP）
- 新增评论/留言后台审核管理
- 后台全面中文化
- 优化数据库索引（posts、comments 表复合索引）
- 优化 BlogController 查询性能
- 时区改为 Asia/Shanghai，locale 改为 zh_CN

### v1.3.1 (2026-04-23)
- 新增文章分页
- 新增代码块复制按钮
- 评论/留言邮箱改为非必填
- 图片上传限制提升至 10MB
- 自定义分页组件（支持页码跳转）
- 修复移动端分页换行问题
- 修复复制按钮 HTTP 环境兼容性问题

### v1.3.0 及更早
- 基础博客功能（文章、分类、标签）
- Filament v3 后台管理
- 评论与留言板系统
- 文章搜索功能
- 基于 Session 的阅读数统计
- 全站设置管理
- 响应式前端界面（Tailwind CSS + Alpine.js）
