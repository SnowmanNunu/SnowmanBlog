# SnowmanBlog 开发路线图

## 版本历史

### v1.4.1 (2026-04-25)
- 修复评论/留言邮箱非必填数据库兼容性问题
- 修复后台中文化字段标签显示问题
- 修复博主自己回复仍需审核的 bug
- 修复留言板后台页面 500 错误
- 评论邮件通知使用正确的管理员邮箱

### v1.4.0 (2026-04-25)
- 新增 Sitemap 功能 (/sitemap.xml)
- 新增 RSS 订阅功能 (/rss.xml)
- 新增文章 SEO 自定义（Meta 标题、描述、关键词）
- 新增文章上一篇 / 下一篇导航
- 新增文章相关推荐
- 新增文章定时发布（Laravel Schedule + Cron）
- 新增评论邮件通知（QQ 邮箱 SMTP）
- 新增评论/留言后台审核管理
- 后台全面中文化
- 优化数据库索引（posts、comments 表复合索引）
- 优化 BlogController 查询性能
- 时区改为 Asia/Shanghai，locale 改为 zh_CN

### v1.3.1 (2026-04-23)
- 新增文章分页
- 新增代码块复制按钮
- 评论/留言邮箱改为非必填
- 图片上传限制提升至 10MB
- 自定义分页组件（支持页码跳转）
- 修复移动端分页换行问题
- 修复复制按钮 HTTP 环境兼容性问题

### v1.3.0 及更早
- 基础博客功能（文章、分类、标签）
- Filament v3 后台管理
- 评论与留言板系统
- 文章搜索功能
- 基于 Session 的阅读数统计
- 全站设置管理
- 响应式前端界面（Tailwind CSS + Alpine.js）
