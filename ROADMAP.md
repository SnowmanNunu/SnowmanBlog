# SnowmanBlog 开发路线图

记录项目开发进度与后续规划。

## 已完成

- [x] 博客基础架构搭建（Laravel 12 + Filament v3 + MySQL 5.7）
- [x] 文章管理系统（发布、编辑、分类、标签、封面图）
- [x] 前端页面（首页列表、文章详情、分类筛选、标签筛选）
- [x] 留言板模块（访客留言 + 博主后台回复）
- [x] 文章评论系统（嵌套回复）
- [x] 站点基础配置（博客标题、描述、ICP 备案号）
- [x] 全局实时搜索（弹窗、300ms防抖、关键词红色高亮、ESC关闭、Ctrl+K快捷键）
- [x] UI 全面优化（导航栏毛玻璃、搜索按钮胶囊样式、搜索弹窗动画、Footer双栏）
- [x] 后台文章编辑器升级为富文本（图文混排、代码块、链接）
- [x] 文章详情页支持 Markdown 解析与代码块样式渲染
- [x] 后台仪表盘数据（统计卡片、30天趋势图、最新文章列表）
- [x] README.md 技术栈介绍
- [x] 文章详情页右侧目录导航（根据 H2/H3 自动生成，点击跳转，滚动高亮）
- [x] 代码块语法高亮（接入 highlight.js github-dark 主题）
- [x] 移动端汉堡菜单（下拉抽屉菜单，支持搜索入口）
- [x] 文章浏览量统计（数据库字段 + session 防刷机制）
- [x] 后台仪表盘待审核评论/留言快捷入口（新提交默认待审核）
- [x] 富文本编辑器图片限制（封面图 2MB/1200px，编辑器上传 2MB 限制）
